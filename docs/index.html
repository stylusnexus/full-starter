What you get

Everything a production team uses to ship software with AI coding agents, in one fork-ready repo.

Development Infrastructure

  • CLAUDE.md — project brain with rules, commands, and earned gotchas
  • Domain guidances — AI safety, database patterns, testing strategy (loaded on demand, not always)
  • Specialized agents — backend, frontend, UX, AI, product, docs, and a domain template
  • Agent memory — persistent knowledge that survives across sessions
  • Workflow skills — brainstorm, TDD, verify, plan, execute, debug, review, deploy
  • Automated hooks — context injection, test guards, coverage checks

Testing Infrastructure

  • AI mock fixtures — deterministic test data instead of expensive live AI calls
  • Auth and profile mocking — test every subscription tier without real accounts
  • Visual verification — screenshot baselines catch what assertions miss
  • Verify loop — build + test + lint before agents claim "done"
  • Experiment-as-test — quality regression detection with baseline comparison
  • Tiered CI — smoke on PR, regression nightly, visual on UI changes (all manual-trigger by default)

These patterns come from a team that ships a production SaaS with AI agents every day. Not theory — patterns extracted from real failures, real fixes, and real workflow evolution over months of daily use.
